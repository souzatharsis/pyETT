# pyETT
Python API for Eleven VR Table Tennis data.

![Eleven VR](https://cdn.akamai.steamstatic.com/steam/apps/488310/capsule_616x353.jpg)

## What's available

- [x] User-profile
- [x] User elo-history
- [x] User friends list
- [x] User-matches (ranked, limit: 25)
- [x] ETT leaderboard (limit: 10)
- [x] User search
